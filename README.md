# SetFrame

Personal studio site for SetFrame (websites, content systems, and business automation).

Built with Next.js (App Router), TypeScript, and Tailwind CSS. See
[implementation_plan_SF.md](./implementation_plan_SF.md) for the full design
and build plan.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Deploy

Deployed via GitHub → Vercel auto-deploy. Target domain: setframe.net.
